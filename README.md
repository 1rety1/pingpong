# 🕹️ PingPong (Pygame)

A modern version of the classic **Pong** game built with Python and Pygame.

## 🎮 Features

* 🤖 Play against AI
* 👥 Local Player vs Player mode (W/S and Arrow keys)
* ✨ Particle effects on ball collisions
* 🎨 Simple UI with styled buttons and menu
* ⏸ Pause menu (ESC)
* 🔄 Restart and return to main menu
* 🏆 Score tracking with saved records

## 🕹 Controls

### Player 1

* `W` – Move up
* `S` – Move down

### Player 2

* `↑` – Move up
* `↓` – Move down

### General

* `ESC` – Pause / Resume
* Mouse – Menu navigation

## 🚀 Installation

1. Make sure you have Python installed
2. Install Pygame:

```bash
pip install pygame
```

3. Run the game:

```bash
python main.py
```
